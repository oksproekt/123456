  2. Настроить маршрутизатор Merkulova

    3. Нажимаем на изображение маршрутизатора. Выбираем вкладку CLI. Нажимаем Enter и набираем следующую последовательность команд для конфигурирования интерфейсов маршрутизатора:

    //для входа в маршрутизатор
    Router>enable
    //для входа в режим конфигурирования маршрутизатора
    Router#config terminal
    Enter configuration commands, one per line. End with CNTL/Z.
    // изменение имени хоста
    Router(config)#hostname Merkulova
    Merkulova(config)#
    // выбор интерфейса маршрутизатора
    Merkulova(config)#interface fa0/0
    //установка IP-адреса интерфейса
    Merkulova(config-if)#ip address 192.168.1.1 255.255.255.0
    // запрет выключения интерфейса
    Merkulova(config-if)#no shutdown
    // выход из конфигурирования интерфейса
    Merkulova(config-if)#exit
    // выбор нового интерфейса маршрутизатора
    Merkulova(config)#interface fa0/1
    // установка IP-адреса интерфейса
    Merkulova(config-if)#ip address 172.16.0.1 255.255.0.0
    // запрет выключения интерфейса
    Merkulova(config-if)#no shutdown
    // выход из конфигурирования интерфейса
    Merkulova(config-if)#exit
    //выход из режима конфигурирования маршрутизатора
    Merkulova(config)#exit
    Merkulova#
    %SYS-5-CONFIG_I: Configured from console by console
    //запись текущей конфигурации в память маршрутизатора
    Merkulova#write memory
    Building configuration...
    [OK]

    Создание веб-страниц с использованием HTML и CSS

    CSS (Cascading Style Sheets) показывает браузеру, какие свойства есть у определенных элементов на странице: шрифт текста, размер изображений и блоков.
    HTML-элементы — основа языка HTML. Каждый HTML-документ состоит из дерева HTML-элементов и текста. Каждый HTML-элемент обозначается начальным (открывающим) и конечным (закрывающим) тегом. Открывающий и закрывающий теги содержат имя элемента.
